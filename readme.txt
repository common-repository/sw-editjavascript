テーマエディタで.jsファイルを編集するためのプラグイン。
This is a plugin to edit the JavaScript file in the theme editor. 

== Description ==
**It is being developed by the Japanese. The explanation in English doesn't exist or there is a possibility that the grammar is not accurate.**

**Japanase**

このプラグインは[ScuderiaWeb](http://scuderia-web.com/)によって開発された、テーマエディタでJavaScriptファイル（.jsファイル）を編集するためのプラグインです。

* テーマエディタにJavaScriptファイルを表示。テンプレートファイルや、CSS同様、編集することができます。
* テーマエディタのファイルリストの並び順を設定可能。ファイル名順、表示名順のいずれかを設定できます。
* テーマエディタのファイルリストに表示する名前を、Template Nameとは別に設定可能。テンプレートファイル内で「File Name: 」として設定したファイル名をテーマエディタに表示。投稿やページの「ページテンプレート」には表示させたくない共通ファイルも、判りやすい名前で表示することができます。

**English**

This is a plugin to edit the JavaScript file in the theme editor. 

* The JavaScript file is displayed in the theme editor. It is possible to edit it as well as the template file and CSS.
* The order of the row of the file list of the theme editor can be selected from either the order of the file name or the order of the display name. 
* The name displayed in the theme editor can be set besides Template Name. The file name set as "File Name: " in the template file can be displayed in the theme editor. 


== Installation ==
1. ダウンロード後解凍し、sw_edit_js.phpの先頭で設定を行います。
2. ファイルをWordPressのpluginsフォルダにアップロードします。
2. 管理画面の「プラグイン」からSW EditJavaScriptを有効化します。
3. テーマファイルで関数をコールします。
